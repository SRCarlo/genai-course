## 2. What is Python?

**Answer:**

Python is a high-level, interpreted, and easy-to-learn programming language created by Guido van Rossum.

### Features

* Easy syntax
* Beginner friendly
* Open source
* Cross-platform
* Large community support

### Example

```python
print("Hello World")
```

---

## 3. What is a Variable?

**Answer:**

A variable is a container used to store data values in memory.

### Example

```python
name = "Shubham"
age = 24
```

### Benefits

* Store information
* Reuse data
* Make programs dynamic

---

## 4. What are Data Types?

**Answer:**

Data types define the type of value stored in a variable.

### Common Data Types

```python
name = "Shubham"     # String
age = 24             # Integer
salary = 50000.50    # Float
is_active = True     # Boolean
```

---

## 5. What is a List?

**Answer:**

A list is an ordered and mutable collection used to store multiple items.

### Example

```python
skills = [
    "Python",
    "AI",
    "Node.js"
]
```

### Features

* Ordered
* Mutable
* Allows duplicate values
* Indexed

### Common Methods

```python
append()
remove()
insert()
pop()
sort()
```

---

## 6. What is a Tuple?

**Answer:**

A tuple is an ordered and immutable collection.

### Example

```python
cities = (
    "Pune",
    "Mumbai"
)
```

### Features

* Ordered
* Immutable
* Faster than lists
* Allows duplicates

---

## 7. What is a Set?

**Answer:**

A set is an unordered collection of unique values.

### Example

```python
skills = {
    "Python",
    "AI",
    "Node.js"
}
```

### Features

* No duplicate values
* Unordered
* Mutable

---

## 8. What is a Dictionary?

**Answer:**

A dictionary stores data as key-value pairs.

### Example

```python
user = {
    "name": "Shubham",
    "age": 24
}
```

### Features

* Key-value structure
* Mutable
* Fast lookup
* Unique keys

---

## 9. Difference Between List and Tuple

| List        | Tuple       |
| ----------- | ----------- |
| []          | ()          |
| Mutable     | Immutable   |
| Slower      | Faster      |
| More memory | Less memory |

### Example

```python
skills = ["Python", "AI"]

cities = (
    "Pune",
    "Mumbai"
)
```

## 10. What is a Function?

**Answer:**

A function is a reusable block of code used to perform a specific task.

### Example

```python
def greet(name):
    return f"Hello {name}"

print(greet("Shubham"))
```

### Benefits

* Reusable code
* Cleaner programs
* Easy maintenance

---

## 11. What is a Loop?

**Answer:**

A loop is used to execute a block of code repeatedly.

### Types of Loops

#### For Loop

```python
for i in range(5):
    print(i)
```

#### While Loop

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

### Benefits

* Reduces repetitive code
* Saves time
* Automates tasks

---

## 12. What is an If Statement?

**Answer:**

An if statement is used for decision-making in Python.

### Example

```python
age = 20

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

### Types

* if
* if-else
* if-elif-else

---

## 13. What is Input in Python?

**Answer:**

The input() function is used to take data from the user.

### Example

```python
name = input("Enter Name: ")
print(name)
```

---

## 14. What is Type Conversion?

**Answer:**

Type conversion means converting one data type into another.

### Example

```python
age = int(input("Enter Age: "))
```

### Common Functions

```python
int()
float()
str()
bool()
```
