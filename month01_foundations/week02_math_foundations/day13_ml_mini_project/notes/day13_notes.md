# Statistics

Statistics helps us understand data.

Example:

```python
from statistics import mean

hours = [1,2,3,4,5]

print(
    mean(hours)
)
```

Output:

```python
3
```

---

# Mean

Mean is the average value.

Formula:

Mean =
Sum of Values ÷ Total Values

Example:

```python
1 + 2 + 3 + 4 + 5 = 15

15 / 5 = 3
```

Mean = 3

# Error

Error is the difference between actual and predicted values.

Formula:

Error =
|Actual - Predicted|

Example:

```python
actual = 75

predicted = 80

error = abs(
    actual -
    predicted
)

print(error)
```

Output:

```python
5
```
