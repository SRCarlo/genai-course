# THEORY — DAY 2 COMPLETED TOPICS

## 1. Python Introduction
Python is a high-level, interpreted programming language widely used in AI, ML, automation, and backend systems.

Why Python for AI:
- Simple syntax
- Huge AI ecosystem
- Libraries like NumPy, Pandas, PyTorch, TensorFlow

---

## 2. Variables
Variables store data in memory.

Example:
```python
name = "Shubham"
age = 24
```

Key point:
- Python is dynamically typed
- No need to declare type

---

## 3. Data Types

### String (str)
Text data

### Integer (int)
Whole numbers

### Float
Decimal numbers

### Boolean (bool)
True or False

Example:
```python
name = "AI"
age = 25
price = 99.5
is_active = True
```

---

## 4. Strings
Strings are heavily used in AI (prompts, responses, text processing).

Operations:
- upper()
- lower()
- replace()
- split()
- len()

---

## 5. Input Function
Used to take user input.

```python
name = input("Enter name: ")
```

Important:
Input is always string type.

---

## 6. Type Conversion
Convert data types:

```python
int("10")
float("10.5")
str(10)
```

---

## 7. Print & f-strings
Used for formatted output.

```python
name = "AI"
print(f"Hello {name}")
```

# INTERVIEW QUESTIONS WITH ANSWERS (DAY 2)

## 1. What is Python?
Python is a high-level programming language used in AI, ML, web development, and automation due to its simple syntax and rich libraries.

---

## 2. Why is Python used in AI?
Because it has powerful libraries like NumPy, Pandas, TensorFlow, PyTorch, and easy integration with AI frameworks.

---

## 3. What is a variable in Python?
A variable is a container used to store data values in memory.

---

## 4. What are Python data types?
Python supports:
- str (text)
- int (numbers)
- float (decimal)
- bool (true/false)

---

## 5. Difference between input() and print()?
input() takes user input  
print() displays output

---

## 6. Why is type conversion needed?
Because input() always returns string, so we convert it to int/float for calculations.

---

## 7. What are strings used for in AI?
Strings are used for prompts, responses, and text processing in LLM applications.

---

## 8. What is f-string?
A formatted string used for injecting variables into text easily.

---

## 9. What is dynamic typing?
Python automatically assigns data types without explicit declaration.

---

## 10. What happens if type conversion is not used?
It causes errors during mathematical operations.
